# Contributing to Global Parikrama Harīm Website

Thank you for your interest in contributing to the Global Parikrama Harīm website!

## How to Contribute

### Types of Contributions

We welcome various types of contributions:

1. **Content Improvements**
   - Clarifying text and messaging
   - Correcting errors or outdated information
   - Adding missing content
   - Improving accessibility

2. **Design & Layout**
   - Responsive design improvements
   - Visual enhancements
   - CSS refinements
   - User experience improvements

3. **Technical Enhancements**
   - Performance optimizations
   - Browser compatibility fixes
   - Accessibility features
   - SEO improvements

4. **Translations**
   - Translating content to other languages
   - Creating multilingual support

5. **Media & Graphics**
   - Photography from our journey
   - Artwork and illustrations
   - Icons and visual elements

### Contribution Process

1. **Fork the Repository**
   ```bash
   git clone https://github.com/b9Joker108/global_parikraama_hariim_website.git
   cd global_parikraama_hariim_website
   ```

2. **Create a Branch**
   ```bash
   git checkout -b feature/your-improvement-name
   ```

3. **Make Your Changes**
   - Test locally by opening HTML files in your browser
   - Ensure responsive design works on mobile devices
   - Check accessibility with screen readers if possible

4. **Test Your Changes**
   ```bash
   # Use a local server for testing
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

5. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Brief description of changes"
   ```

6. **Push and Create Pull Request**
   ```bash
   git push origin feature/your-improvement-name
   ```
   Then create a pull request on GitHub

### Guidelines

#### Code Style
- **HTML**: Use semantic HTML5 elements
- **CSS**: Follow existing naming conventions and organization
- **JavaScript**: Write clear, commented code
- **Formatting**: Use consistent indentation (4 spaces)

#### Content Guidelines
- **Tone**: Compassionate, inclusive, welcoming
- **Language**: Clear, accessible, avoiding jargon
- **Accuracy**: Ensure alignment with our values and practices
- **Respect**: Honor all identities, traditions, and paths

#### Accessibility
- Use semantic HTML tags
- Include alt text for images
- Ensure sufficient color contrast
- Make interactive elements keyboard accessible
- Test with screen readers when possible

#### Performance
- Optimize images before adding
- Minimize CSS and JavaScript
- Avoid large external dependencies
- Test load times

### What to Contribute

#### High Priority
- Accessibility improvements
- Mobile responsiveness enhancements
- Performance optimizations
- Content clarifications
- Translation support infrastructure

#### Medium Priority
- Visual design refinements
- Additional pages or sections
- Interactive features
- Blog/news section
- Photo galleries

#### Nice to Have
- Advanced animations
- Enhanced navigation
- Search functionality
- Comments or feedback forms

### Review Process

1. **Initial Review**: Maintainers review for alignment with project goals
2. **Technical Review**: Check code quality and functionality
3. **Content Review**: Ensure messaging aligns with community values
4. **Testing**: Verify changes work across browsers and devices
5. **Approval**: Requires approval from Principal Visionary for major changes

### Areas Needing Help

- **Photography**: High-quality images of spiritual sites and community life
- **Translations**: Spanish, Hindi, Thai, Tibetan, and other languages
- **Accessibility**: WCAG 2.1 AA compliance testing and improvements
- **Documentation**: Adding code comments and documentation
- **Testing**: Cross-browser and device testing

## Alignment with Values

All contributions must align with our core principles:

- **Compassion**: Kind, supportive language and design
- **Inclusivity**: Welcoming all identities and backgrounds
- **Sustainability**: Efficient, lightweight code
- **Accessibility**: Ensuring all can access our message
- **Beauty**: Aesthetic expression of spiritual values
- **Respect**: Honoring Buddha Maitreya's visionary authority
